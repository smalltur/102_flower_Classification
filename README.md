This is a 102-flower classification model, with the dataset sourced from https://www.robots.ox.ac.uk/~vgg/data/flowers/102/index.html
After downloading the data, divide the files into three folders, named train test and val.
Replace their file paths with the file paths in "train.py" and "predict.py"
Then you can run the "train.py" file and "predict.py"
